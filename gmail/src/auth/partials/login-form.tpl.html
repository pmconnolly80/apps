Login

Why no login form?

Since we're using OAuth to authenticate the user, submitting login information is done at the OAuth provider (i.e. Google). The user provides his credentials to Google and Google returns an auth token if authentication and authorization were successful.

Our app never knows the user credentials and shouldn't provide a login form, just a button to trigger the OAuth sequence.

Warning! You're viewing the Login screen while being already authenticated.

You can go directly to the Message List, but in a real application, you would probably redirect the user automatically. To do so, you could use the following code:

$rootScope.$on('$stateChangeStart', function (event, next) {
  if (next.name == 'login' && AuthService.isAuthenticated()) {
    $state.go('messages');
  }
});
